Database Commands

Command Description

create database if not exists

 commands are case insensitive, and must end with a comma

show "database"

this will show all databases on the SQL server

use "database"

this will select and use a specific database on the SQL server

Table Column Commands

Command Description

CREATE TABLE IF NOT EXISTS table-name (col type , col type);

This command creates a table in the chosen database

EXPLAIN table-name;

This commands shows the format of any table

DROP TABLE IF EXISTS table-name;

This command deletes a table from the database

SOURCE path of SQL file

accesses an SQL script file from command prompt

Alter TABLE table-name

Adds, deletes, or modifies columns in a table

TABLE DATA COMMANDS

Command Description

INT

An integer -2,147,483,648 to 2,147,483,648

DECIMAL

A floating point number thay can specify the number of permissible digits. Example: (5,2) allos -999.99 to 999.99

DOUBLE

A double-precision floating point number

DATE

A date om the YYYY-MM-DD format

TIME

A time in the HH:MM:SS format

DATETIME

A combined date and time in the format YYYY-MM-DD HH:MM:SS

YEAR

A year 1901-2155 in either YY or YYYY format

TIMESTAMP

Automatic date and time of last record entry

CHAR()

A string of defined fixed length up to 255 characters with padding. CHAR(50) for example would limit string to 50 scharacters.

VARCHAR()

A string of variable length up to 255 characters without padding

TEXT

A string up to 65,535 characters long

BLOB

A binary type for variable data

ENUM

A single string value from a defined list. Example: ENUM ("blue", "yellow", "green") allows entry of only those clolors

SET

A string of multiple strings from a defined list.

NOT NULL

Insists that each record must include a data value in a column

UNIQUE

Insists that records may not duplicate any entry in this column.

AUTO_INCREMENT

Automatically generates a number that ius one more than the previous numeric column

DEFAULT

Specifies a a value to be used where no value is started for this column when a record is inserted

PRIMARY KEY

Specifies the column/columns to be used as the primary key for that table
